# imigz
   imigz is an image sharing and image hosting service
  
## Details
  

## Setup Dev enviroment
```bash
  npm install
  docker-compose up -d
  npm run migrate:latest
```
## Run app 
```bash
  npm run build
  npm run start
  # or
  npm run dev
```
## Run tests

```bash
  # Make sure first that containers are up and latest migration is done
  npm run test
```
